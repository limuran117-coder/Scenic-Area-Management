Scenic-Area-Management
景区营销数据管理与分析工作空间
项目简介
这是一个专注于景区营销数据管理与分析的AI工作空间，主要服务于建业电影小镇的日常运营与决策支持。
核心功能
数据监测与分析
• 抖音指数日报 - 每日竞品景区抖音热度监测
• 小红书数据日报 - 品牌声量与KOC效果分析
• 客流数据管理 - 实时客流统计与分析
营销工具
• GEO资料库 - 问答知识库生成与发布
• 竞品监控 - 只有河南、银基动物王国、万岁山武侠城等竞品动态
• 活动策划 - 主题活动、发布会等营销支持
定时任务
任务	时间	说明
天气日报	7:00	每日天气速报
抖音指数日报	8:00	竞品抖音热度
小红书日报	10:00	品牌数据洞察
