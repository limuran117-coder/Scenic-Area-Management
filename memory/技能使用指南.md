python3 -c "
import csv
with open('数据文件.csv', 'r') as f:
    reader = csv.DictReader(f)
    for row in reader:

"

### 🌐 网页爬取

#### Scrapling（已配置环境）
```bash
# 激活环境
source ~/.scrapling-venv/bin/activate

# 简单网页抓取
scrapling extract get "https://example.com" output.md

# 动态内容
scrapling extract fetch "https://example.com" output.md

# 反爬虫绕过
scrapling extract stealthy-fetch "https://example.com" output.md

# 跳过SSL验证
scrapling extract get "https://example.com" output.md --no-verify
```

## 四、电影小镇工作流

### 1. 每日数据日报
```
8:00 抖音指数日报 → douyin_index_v9.py
10:00 小红书日报 → xiaohongshu_report.py --daily
```

### 2. 竞品监控
```
每周 → 抓取竞品网站数据 → scrapling
     → 分析 → csv-pipeline
     → 报告 → xiaohongshu_report.py
```

### 3. GEO资料更新
```
搜集资料 → 整理 → feishu-doc-manager
       → 发布到飞书
```
