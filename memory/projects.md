## 📊 数据三维度

### 维度1：往年数据
- **来源**：`2023-2025年门票销售及客流统计数据表.xlsx`
- **用途**：历史对比、趋势分析
- **调用方式**：`query_data_v2.py --type month/holiday`

### 维度2：26年计划任务目标
- **来源**：`电影小镇2026年经营计划.md` / 2026年经营数据模板
- **用途**：目标追踪、进度对比
- **调用方式**：引用计划文档

- **来源**：`~/Desktop/2026年电影小镇实际客流.xlsx`
- **用途**：实时进度、实际完成情况
- **调用方式**：读取实际客流Excel

## 🔄 常用查询命令

### 查询往年数据
```bash
python3 scripts/query_data_v2.py --type month --month 1
python3 scripts/query_data_v2.py --type holiday --name 春节
python3 scripts/query_data_v2.py --type validate
```

### 读取2026年实际数据
```bash
# 桌面文件
~/Desktop/2026年电影小镇实际客流.xlsx
```

### 读取计划目标
```bash
# 文档文件
电影小镇2026年经营计划.md
```
