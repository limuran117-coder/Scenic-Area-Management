## 三、核心方法论提炼

### 1. 有机增长 playbook (b2c-marketing)
- 六阶段增长系统
- 账号预热 → 内容测试 → 爆款识别 → 规模化
- Hook公式 + CTA策略 + 转化追踪

### 2. 冷外展框架 (cold-outreach)
- 个性化 + 价值主张 + 行动号召
- 邮件/SMS/LinkedIn多渠道

### 3. 内容营销漏斗
- Lead Magnet → Landing Page → 转化
- SEO驱动 + 社交放大 + 邮件培育
