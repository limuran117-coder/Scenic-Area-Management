# HEARTBEAT.md - 主动检查任务

## 任务说明
每次 Heartbeat 轮询时，轮流执行以下任务（不要一次执行多个）

### 1. 邮件检查（周一-周五）
- 检查是否有重要未读邮件
- 如果有重要邮件，发送到群提醒

### 2. 定时任务状态检查（每天）
- 检查今日待执行的定时任务
- 如果有任务失败风险，发送预警

### 3. 记忆整理（周三、六）
- 检查 memory/ 目录
- 将重要的 daily notes 整理到 MEMORY.md

### 4. 服务状态检查（每天）
- 检查 Docker 容器运行状态
- 检查 TrendRadar 是否正常
- 如果异常，发送告警

### 5. 学习资料更新（周二、四）
- 检查是否有新的行业报告/白皮书
- 标记待学习内容

### 6. 竞品热点动态监控（每天）
- 监控方式：监测抖音指数/小红书数据异动
- 监控对象：只有河南、银基动物王国、万岁山武侠城
- 发现数据异动 → 分析原因 → 及时应对

### 7. GEO资料更新（每天）
- 搜集竞品动态/营销案例
- 改编为电影小镇GEO问答
- 质量检查：基础信息、标配信息源
- 更新到GEO资料库
